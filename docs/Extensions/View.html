  • applyThemeButtonStyle()

    Applies a complete themed button style using values from the environment or theme defaults.

    This should be the base modifier applied to any Button that wants to use the theme system. It will apply the variant, size, and shape defined either explicitly or from Theme.buttons.

    It should be called before other modifiers like .buttonVariant(...).

  • applyThemeTextFieldStyle()

    Applies the theme’s default text field style and propagates environment configuration like variant, size, shape, and font.

    Call this before other style-specific modifiers like .textFieldVariant(_) or .textFieldFont(_).

    Example:

  .applyThemeTextFieldStyle()

  • navigationTitle(_:style:weight:)

    Sets a custom navigation title using the theme’s typography system.

    This modifier allows you to style the navigation bar title with a specific text style from your theme, and optionally apply a font weight.

    Example:

    NavigationStack {
  MyView()
    .navigationTitle("Settings", style: .titleLarge, weight: .semibold)
}

    Declaration

    Swift

    @MainActor
func navigationTitle(
  _ title: String,
  style: TextStyleToken = .headlineLarge,
  weight: Font.Weight? = nil
) -> some View

    Parameters

    title

    The string to display as the navigation title.
    style

    The typography style token from your theme (e.g., .headlineLarge, .titleMedium). Defaults to .headlineLarge.
    weight

    An optional font weight override (e.g., .bold, .medium).

    Return Value

    A view with a themed navigation title applied.
